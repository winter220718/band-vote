# 합주곡 투표 웹사이트

모바일 최적화된 합주곡 투표 사이트입니다.

## 주요 기능
- 이름 입력 후 투표 페이지 이동
- 곡명 + 유튜브 링크 목록 표시
- 체크박스로 복수 선택 후 제출
- 모바일 반응형 UI
- 관리자 화면에서 선곡 추가, 수정, 삭제
- 관리자 화면에서 투표 결과 및 제출 데이터 확인


## 무료 웹 배포 추천
IP 입력 없이 공개하려면 Render 같은 무료 웹호스팅이 가장 간단합니다.

### Render 배포 방법
1. 이 프로젝트를 GitHub 저장소에 올립니다.
2. Render에서 New + > Blueprint 또는 Web Service를 선택합니다.
3. 저장소를 연결하면 [render.yaml](render.yaml) 설정으로 바로 배포할 수 있습니다.
4. 배포가 끝나면 Render가 공개 URL을 발급합니다.

### 참고 사항
- 무료 플랜은 일정 시간 미사용 시 잠들 수 있습니다.
- 현재 데이터는 SQLite 파일로 저장됩니다.
- 배포 환경에 따라 SQLite 파일은 재배포 시 초기화될 수 있으니, 장기 운영은 별도 DB를 권장합니다.
